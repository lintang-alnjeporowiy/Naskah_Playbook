Peraturan Main Google Drive Buku iStow

1. Draft tulisan atau naskah dapat dimasukkan ke dalam folder "Draft" sesuai dengan Bab Masing-Masing atau langsung masukkan di Google Docs di setiap folder babnya, untuk penempatannya saat di tata letak bisa digambarkan melalui Google Docs yang tersedia pada folder bab masing-masing.
2. Referensi berupa buku, artikel dan material presentasi dapat dimasukkan kedalam folder "Referensi", karena sifatnya umum maka tidak ada kategorisasi disana, just drop it.
3. Material berupa gambar, grafik atau data mentah mohon dimasukkan kedalam folder "Resources" dan diberi judul yang jelas, panjang ga papa, use all the 256 bit that we got.
4. Naskah atau tulisan yang selesai di edit dapat dilihat di https://github.com/lintang-alnjeporowiy/Naskah_Playbook
5. Bab yang sudah selesai tata letak dapat dilihat di folder "Layouted"
